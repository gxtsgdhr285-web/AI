## **Introduction**

**TrendBot** is an automated intelligence pipeline that collects, analyzes, and summarizes the latest overseas startup trends. It aggregates data from multiple sources to generate a unified **Weekly Overseas Startup Trend Report**.

**Data Sources:**

- **ProductHunt** – Daily Top 10 products
- **TechCrunch Startups** – Latest funding & startup news
- **Reddit** – `/r/startups` & `/r/sideproject` top weekly discussions
- **ExplodingTopics** – Rapidly emerging search trends
- **Y Combinator** – Monthly new startups
- **Perplexity API** – For additional intelligence extraction

TrendBot is designed for:

- Entrepreneurs looking for inspiration from overseas startups
- Research teams tracking emerging startup patterns
- VC analysts monitoring global market signals
- Makers, builders, and AI product designers

---

## **Features**

- Automatic weekly collection of top startup trends
- Data aggregation from multiple platforms (RSS + web scraping)
- Intelligent summarization using OpenAI API
- Trend scoring and ranking
- Optional sync to **Notion** for internal team tracking
- Scheduled automation via GitHub Actions
